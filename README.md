# Sistema de Gestión de Inventarios con Polimorfismo

Este proyecto demuestra el uso de **polimorfismo**, **clases abstractas** e **interfaces** en Java.
Permite manejar diferentes tipos de productos (Electrónicos, Alimentos y Ropa) bajo una misma estructura de inventario.
